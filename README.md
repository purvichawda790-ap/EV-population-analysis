 Electric Vehicle Population Dashboard - Tableau Project

 Project Overview

Developed an interactive Tableau dashboard to analyze electric vehicle population data across different regions, vehicle types, and manufacturers. The dashboard provides insights into EV adoption trends, popular vehicle brands, and growth patterns using interactive visualizations and filters.

 Tools & Technologies Used

* Tableau Public
* Excel / CSV Dataset

 Dashboard Features

* Interactive filters for detailed analysis
* EV population trends visualization
* Manufacturer-wise vehicle analysis
* Regional distribution insights
* KPI cards and comparative charts

 Key Insights

* Electric vehicle adoption has increased significantly over recent years
* Certain manufacturers dominate the EV market share
* Regional analysis highlights areas with higher EV penetration

 Files Included
